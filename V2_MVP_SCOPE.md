# V2 MVP Scope

## Included

1. Home / workflow search
2. Quick Note
3. Detailed Encounter
4. Medical Report / Letter
5. Feedback
6. Safety / About

## Intentionally excluded from first MVP

- legacy calculator experience in the main navigation
- backend or login
- workflow approval promotion
- clinical decision support features
- public deployment automation

## MVP success criteria

- Vite app runs locally
- 1,500 workflows load
- 12 exclusions are hidden by default
- direct access to excluded workflows is blocked
- Quick Note works for a selected workflow
- Detailed Encounter basic workflow-specific structure works
- production build passes
