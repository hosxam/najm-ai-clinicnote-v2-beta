# Beta Tester Message

Hello team — this is the limited beta testing build of **Najm ClinicNote V2**.

Please note:
- This build is **not clinically approved**.
- This build is **not clinical decision support**.
- Use **mock or anonymized cases only**.
- **Do not enter patient identifiers**.
- Please start with **common low-risk workflows first**.
- All outputs are **clinician-review drafts** and must be reviewed before any use.

While testing, please report any of the following:
- unsafe wording
- invented content
- missing fields
- confusing UI behavior
- workflow search problems
- hidden/blocking issues with excluded workflows

Thank you — your feedback will help us improve safety, clarity, and usability before any broader internal rollout.
