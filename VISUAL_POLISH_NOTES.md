# Visual Polish Notes

## Scope

This pass updates presentation only. It does not change:

- imported workflow data
- exclusion configuration
- note-generation logic
- safety rules
- product behavior

## Visual improvements

- Calmer background gradient and softer card shadows
- Clearer hierarchy in the top header and section cards
- More compact but cleaner safety banner
- Better workflow search card scanability
- Stronger selected and hover states for workflow cards
- More consistent field styling for inputs, textareas, and selects
- Cleaner output panels and tab/button grouping
- Better spacing across Home, Quick Note, Detailed Encounter, Medical Report, Feedback, and Safety/About
- Improved mobile-friendly spacing and reduced visual clutter

## Still visually weak

- There is no custom iconography or brand illustration system yet
- Output text is functional but still intentionally plain
- Some long workflow titles may still wrap heavily on very narrow screens
- No formal visual regression testing is in place yet
